Popup Panic Button

Enable Popup Panic Button
Popup Side: Left Right
Enabling the Popup Panic Button will allow the extension to insert a popup version of the panic button into the corner of any webpage you visit, providing you with immediate access to the motivational material you need!
You may "blacklist" specific sites where the popup box won't appear. Add any site addresses you wish to hide the popup box on to the text box below, separated by semi-colons. The blacklist supports the use of wild-cards, simply place an asterisk (*) as a placeholder for part of the address and the plugin will not use the popup panic button on any cases that match that address.
Valid ways to blacklist sites and specific pages include:
  • www.example.com
  • example.com
  • http://www.example.com/specific/page
  • *.example.com
